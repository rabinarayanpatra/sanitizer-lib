In Action

One annotation. Done.

Annotate any field with @Sanitize and chain one or more sanitizers. The framework intercepts Jackson deserialization for DTOs and JPA lifecycle events for entities — no manual calls, no missed fields.

UserDto.java 
public class UserDto {

  @Sanitize(using = {
    TrimSanitizer.class,
    LowerCaseSanitizer.class
  })
  private String email;

  @Sanitize(using = CreditCardMaskSanitizer.class)
  private String cardNumber;

  // getters & setters
}
Install

Add it to your build

Available on Maven Central. Spring Boot 3.4+ and Java 21.

build.gradle.kts 
// Spring Boot integration
implementation("io.github.rabinarayanpatra.sanitizer:sanitizer-spring:1.1.0")

// JPA integration (optional)
implementation("io.github.rabinarayanpatra.sanitizer:sanitizer-jpa:1.1.0")
pom.xml 
<dependency>
  <groupId>io.github.rabinarayanpatra.sanitizer</groupId>
  <artifactId>sanitizer-spring</artifactId>
  <version>1.1.0</version>
</dependency>
